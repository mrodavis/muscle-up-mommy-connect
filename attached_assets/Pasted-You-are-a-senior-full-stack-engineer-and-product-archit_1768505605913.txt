You are a senior full-stack engineer and product architect.

Your task is to design and scaffold a production-grade MVP for a mobile-first social platform called **Muscle Up Mommy Connect**.

This platform is conceptually similar to **AfroTech Connect**, but built for moms and maternal wellness, with integrated community, fitness programs, events, and in-app commerce.

==============================
CORE PRODUCT GOAL
==============================
Build a scalable social + fitness + events + commerce platform that allows Muscle Up Mommy to operate as a full ecosystem, not just a brand.

The system must support:
- Community networking
- Membership tiers
- Fitness program management
- Events & meetups
- Gym discovery
- In-app store integration
- AI-ready customer support and coaching hooks

This is NOT a demo app. It must be architected for real users and future scale.

==============================
TECH REQUIREMENTS
==============================
Frontend:
- React Native (Expo preferred)
- Mobile-first design
- Component-driven architecture

Backend:
- Node.js + Express
- API-first architecture

Database:
- PostgreSQL
- Prisma ORM

Auth:
- Email + password authentication
- JWT-based auth
- Role-based access (user, coach, admin)

General:
- Modular, scalable codebase
- Environment variables for secrets
- Clean, commented, production-ready code
- No hardcoded secrets
- No mock-only shortcuts

==============================
BRANDING & UI SYSTEM (REQUIRED)
==============================
The application must strictly follow the Muscle Up Mommy product brand theme.

PRIMARY COLORS:
- Black: #0B0B0B (primary background)
- White: #FFFFFF (primary text and contrast)
- Soft Blush Pink: #F6C1CC (primary accent)

SECONDARY / NEUTRAL COLORS:
- Charcoal: #1A1A1A (cards, containers)
- Light Gray: #EDEDED (dividers, borders)
- Muted Rose: #E8AEB7 (badges, highlights only)

DESIGN PRINCIPLES:
- Calm, confident, premium wellness aesthetic
- Feminine but strong
- Dark-mode-first
- Minimalist and uncluttered
- No bright or neon colors
- No gold, yellow, metallics, or gradients
- Avoid default blue/green UI styles

IMPLEMENTATION REQUIREMENTS:
- Create a centralized theme system (theme.ts)
- Export color tokens, spacing, typography
- All components must consume values from the theme
- No hardcoded color values inside components
- Consistent spacing, subtle rounded cards
- Soft pink used sparingly for CTAs and active states

Apply this brand system consistently across:
- Auth screens
- Community feed
- Profiles
- Groups
- Events
- Store
- Fitness Reboot module
